🎯 Target Variable: Unemployment Risk Level

Each state-quarter is classified into three risk levels based on the unemployment rate:

LOW
u_rate < 3.0%
385 records (43%)
MEDIUM
3.0% ≤ u_rate < 4.5%
340 records (38%)
HIGH
u_rate ≥ 4.5%
171 records (19%)
def risk_level(u):
    if u < 3.0:   return 'Low'
    elif u < 4.5: return 'Medium'
    else:          return 'High'

df['risk'] = df['u_rate'].apply(risk_level)

⚙️ Feature Engineering

Lagged features simulate a realistic forecasting scenario — we use previous quarter information to predict current quarter risk level.

df['u_rate_lag1'] = df.groupby('state')['u_rate'].shift(1)
df['p_rate_lag1'] = df.groupby('state')['p_rate'].shift(1)
df['lf_lag1']     = df.groupby('state')['lf'].shift(1)
df['quarter']     = df['date'].dt.quarter
df['year']        = df['date'].dt.year
model_df = pd.get_dummies(model_df, columns=['state', 'quarter'], drop_first=True)

Train/Test split: 80% / 20% with stratification to preserve class balance.
